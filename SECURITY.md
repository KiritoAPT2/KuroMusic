# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of KuroMusic seriously. If you have found a security vulnerability in our application, we appreciate your help in disclosing it to us in a responsible manner.

### Process

1. **Do not** create a public GitHub Issue for security vulnerabilities.
2. Please report the issue by emailing us at **<security@kuromusic.com>** or by opening a private advisory if enabled on this repository.
3. Include a detailed description of the vulnerability and steps to reproduce.

### Response Time

We are committed to addressing security vulnerabilities promptly. You can expect a response within **48 hours** of your report. We will keep you updated on our progress as we investigate and resolve the issue.

Thank you for helping keep KuroMusic safe for everyone!
