**Overview:** KuroMusic es un cliente open-source de Music Pro para Android diseñado con Material Design 3 en tonos negro (#000000) y morado (#7B1FA2). Ofrece una experiencia musical sin anuncios, con reproducción en segundo plano, modo offline y total privacidad. No está afiliado a Google LLC ni a YouTube.

## Tech Stack

| Frontend | Backend | Tools |
|----------|---------|-------|
| Jetpack Compose | TensorFlow | Gradle |
| Material 3 (negro/morado) | | Git |

## Key Features

### Core Functionality

| Feature | Description |
|---------|-------------|
| 🎵 **Ad-free Playback** | Sin interrupciones publicitarias |
| 🔄 **Background Playback** | Continúa escuchando en otras apps |
| 🔍 **Advanced Search** | Encuentra songs, videos, albums y playlists |
| 👤 **Account Integration** | Sync de preferencias y colecciones |
| 📚 **Library Management** | Organiza tu colección completa |
| 📱 **Offline Mode** | Descarga música para escuchar offline |

### Audio Enhancement

| Feature | Description |
|---------|-------------|
| 🎤 **Synchronized Lyrics** | Letras sincronizadas perfectas |
| ⚡ **Smart Silence Skip** | Salta silencios automáticamente |
| 🔊 **Volume Normalization** | Balancea volúmenes entre tracks |
| 🎛️ **Tempo & Pitch Control** | Ajusta velocidad y tono a tu gusto |

### Personalization

| Feature | Description |
|---------|-------------|
| 🎨 **Dynamic Theming** | Se adapta a los colores de las portadas |
| 🌐 **Multi-language** | Disponible en múltiples idiomas |
| 🚗 **Android Auto** | Compatible con autos |
| 🎯 **Material Design 3** | Diseño Google oficial y moderno |
| 🖼️ **Artwork Export** | Guarda imágenes en HD |

**Security:** Descarga solo de fuentes oficiales.

## Building from Source

Requisitos: Gradle 7.5+, Kotlin 1.7+, Android Studio 2022+.

```bash
git clone https://github.com/KuroMusic/KuroMusic.git
cd KuroMusic
./gradlew assembleRelease
```
